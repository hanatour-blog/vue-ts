### 셋팅 완료 ! 이제 과제를 진행 ! 

CRUD 를 만들어 보자.
- [] 할일 입력란 에 할일을 입력후 추가 버튼을 누르면 List 에 할일이 추가되어야 한다
- [] 추가후 입력란 (input) 은 초기화 "" 처리가 되어야 한다.
- [] 추가된 할일 목록중 특정 할일을 선택시 Done (취소선) 기능이 되어야한다.
- [] 삭세 버튼 선택시 선택한 할일이 목록에서 제거되어야 한다.
